# 개인화 대시보드 웹 애플리케이션

## 프로젝트 한 줄 요약
OpenAPI를 활용해 날씨 정보를 불러오고,  
JSON 데이터와 로컬 저장소를 함께 사용해  
**개인화된 위젯 화면을 구성한 대시보드 웹 애플리케이션**입니다.

---

## 진행 기간
- 2025.12.01 ~ 2025.12.04 (3일)

---

## 주요 기능
- 로그인 및 사용자 상태 관리  
- 날씨 정보 제공(OpenAPI 연동)  
- 실시간 시계 구현  
- 할 일(Todo) 관리 기능  
- 타이머 기능  
- 랜덤 명언 표시  

---

## 기술 스택
- React  
- Axios  
- JSON  
- CSS3  

---

## 배운 점

### OpenAPI를 활용한 데이터 연동 경험
외부 OpenAPI를 호출해 데이터를 받아오고,  
응답 값을 가공해 화면에 표시하는  
**기본적인 API 활용 흐름**을 익힐 수 있었습니다.

### OpenAPI 응답 데이터 구조 이해
OpenAPI에서 내려오는 데이터 구조를 확인하고,  
필요한 값만 선택해 화면에 사용할 수 있도록 처리하는 경험을 통해  
**데이터 구조 분석의 중요성**을 이해할 수 있었습니다.
