# My Widget

JSON 데이터와 로컬 저장소를 함께 사용해 개인화된 위젯 화면을 구성한

## 배운 점 / 인사이트

### OpenAPI 활용 경험

외부 OpenAPI를 호출해 데이터를 받아오고,  
응답 데이터를 화면에 맞게 가공해 사용하는 기본적인 API 활용 흐름을 익힐 수 있었습니다.

### 상태 관리 경험

OpenAPI에서 내려오는 데이터 구조를 직접 확인하며,  
필요한 값만 선택해 React 상태로 관리하는 경험을 할 수 있었습니다.
