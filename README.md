# Dashboard

## 한 줄 요약
React 기반으로 구현한 개인 대시보드 웹 애플리케이션으로, 로그인 상태 관리와 함께 날씨, 시계, 할 일, 타이머 등 일상에 유용한 위젯 기능을 제공합니다.

---

## 진행 기간
- 개인 프로젝트  
- 2025.12.01 ~ 2025.12.04(3일)

---

## 주요 기능

### 로그인 및 사용자 상태 관리
- 사용자 이름을 입력해 로그인 가능
- localStorage를 활용해 새로고침 후에도 로그인 상태 유지

### 날씨 정보 표시
- 외부 API를 활용해 현재 위치 기반 날씨 정보 제공

### 실시간 시계
- 현재 시간을 실시간으로 표시하는 디지털 시계 구현

### 할 일(Todo) 관리
- 할 일 추가 및 목록 관리 기능 제공

### 타이머 기능
- 사용자가 설정한 시간에 맞춰 동작하는 타이머 구현

### 랜덤 명언 표시
- 페이지 하단에 랜덤 명언을 표시해 간단한 UX 요소 추가

## 배운 점

- **React 상태 관리 기초 이해**
  - `useState`, `useEffect`를 활용해 사용자 로그인 상태와 컴포넌트 렌더링 흐름을 관리하는 방법을 익혔습니다.

- **localStorage를 활용한 데이터 유지**
  - 브라우저 저장소를 활용해 새로고침 후에도 데이터를 유지하는 방법을 경험했습니다.

- **컴포넌트 분리의 중요성**
  - 날씨, 시계, 할 일, 타이머 등 기능별로 컴포넌트를 분리하며 React 컴포넌트 구조에 대한 이해를 높였습니다.

- **외부 API 연동 경험**
  - Axios를 활용해 외부 API 데이터를 받아와 화면에 표시하는 기본적인 데이터 흐름을 학습했습니다.
